# v1.0.0-alpha (2024-XX-XX)

## Alpha Release

Initial alpha release of Continuous Face Zero Trust platform.

### Features
- Auth API with face-based authentication
- Face ML pipeline with ONNX runtime
- ZK proof generation (Groth16, PLONK)
- Quantum RNG with local simulator
- PQC cryptography (Kyber, Dilithium)
- Redis/DragonflyDB session store
- Qdrant vector database
- Cloudflare edge gateway
- Istio service mesh
- Multi-cloud deployment (K8s)
- OWASP Top 10 security tests
- CI/CD pipelines

### Known Issues
- Quantum backends limited to local simulator
- GPU acceleration not yet available
- Some PQC algorithms need liboqs native bindings
